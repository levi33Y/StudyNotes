# 请求



## useRequest

const { data, error, loading } = useRequest(getUsername);



支持默认/手动触发



### 配置

|              |               |
| ------------ | ------------- |
| manual       | 默认/手动触发 |
| loadingDelay | loadding      |
| refreshDeps  | 依赖          |
| debounceWait | 防抖          |
| throttleWait | 节流          |
